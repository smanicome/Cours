Exercice 1 - Style de base

  1. Ajoutez en utilisant la balise <h2> le titre de l'exercice courant puis ajoutez à la balise h2 un attribut style qui permet de mettre le titre de l'exercice en orange.
  2. Déclarez une balise style à l'intérieur des balises head du document, puis faites en sorte que tout le document utilise une police sans-serif.
    <style>
     html {
      font-family: sans-serif;
     }
    </style>
  3. Faites en sorte que la balise h1 corresponde à une police de 24 points, la balise h2 à une police de 18 points et la balise h3 à une police 16 points.
    h1 {
     font-size: 24;
    }

    h2 {
     font-size: 18;
    }

    h3 {
     font-size: 16;
    }
  4. Faites en sorte que les balises h2 soient affichées en bleu, à part le titre de cet exercice (qui doit rester en orange).
    h2 {
     font-size: 18;
     color: blue;
    }
    ...
    <h2 style="color:orange">Exercice 1 - Style de base</h2>
  5. Faites en sorte que le titre du compte rendu de TP soit centré. Pour cela, vous allez créer une classe "centered" et ajouter un attribut class à la balise englobant le titre du compte rendu. Déclarez ensuite que le style pour la classe "centered" est centré.
    .centered {
     text-align: center;
    }
    ...
    <h2 class="centered" style="color:orange">Exercice 1 - Style de base</h2>
  6. On veut que votre nom soit aligné à droite. Commencez par déclarer votre nom entre des balises div puis ajoutez une classe right permettant d'aligner le texte à droite.
    MANICOME
    .right {
     text-align: right;
    }
    ...
    <div class="right">MANICOME</div>
  7. Ajoutez un lien vers le cours http://forax.github.io/course/progweb/ puis faites en sorte que le lien soit en vert si la souris n'est pas sur le lien et en teal si la souris survole le lien.
    http://forax.github.io/course/progweb/ a {
     color: green;
    }

    a:hover {
     color: teal;
    }
    ...
    <a href="http://forax.github.io/course/progweb/">http://forax.github.io/course/progweb/</a>

Exercice 2 - Style de liste et de table

  2. Pour la suite du TP, nous ne modifierons plus la balise style du compte rendu mais le fichier compte-rendu-tp3.css Faites en sorte que la liste des questions de cet exercice et de l'exercice précédent utilise les chiffres romains (en minuscule) plutôt que les traditionnels chiffres 1, 2, 3, etc.
    ol {
     list-style-type: lower-roman;
    }
  3. Le site https://en.wikipedia.org/wiki/List_of_2018_box_office_number-one_films_in_the_United_States contient un tableau des films au box office. On souhaite recopier les données (en fait, uniquement le rang, le titre du film et les recettes actuelles) pour les 5 films ayant rapporté le plus d'argent (il y a des flèches pour trier) pour les mettre dans une table dont nous allons changer l'aspect. Donc dans un premier temps, créez la table HTML en utilisant les balises table, th et td dans votre compte rendu.
    # Titre Recettes
    17 Avengers: Endgame $357,115,007
    29 The Lion King $191,770,759
    10 Captain Marvel $153,433,423
    18 Avengers: Endgame $147,383,211
    47 Frozen II $130,263,358
  4. Faites en sorte que la table occupe toute la largeur du compte rendu.
    table {
     width: 100%;
    }
  5. Centrez les titres et changer la police de caractères des titres pour utiliser des petites majuscules (small caps).
    thead {
     text-align: center;
     font-variant: small-caps;
    }
  6. Aérez la table avec la propriété padding pour que le texte ne soit pas collé aux bordures.
    td {
     padding: 1em;
    }
  7. Faites en sorte que les valeurs des deux premières colonnes (rang et titre) soient alignées à gauche et que les valeurs de la dernière colonne soient centrées.
    td:first-child, td:nth-child(2) {
     text-align: left;
    }

    td:last-child {
     text-align: right;
    }
  8. Changez la couleur de fond des lignes paires (even) pour mettre du bleu clair en utilisant la pseudo-classe :nth-child.
    Attention la balise <thead> n'est pas prise en compte tr:nth-child(even) {
     background-color: lightblue;
    }
  9. Faites en sorte que si l'on survole une ligne avec la souris la couleur du fond corresponde à 18 sur 255 de rouge, 28 sur 255 de vert et 80 sur 255 de bleu en utilisant un codage hexadécimal. Note: 26 en décimal correspond à 1A en hexadecimal car pour les dizaines, 26 / 16 = 1, et pour les unités 26 - 1 * 16 = 10 (qui s'écrit 'A').
    tr:hover {
     background-color: #121c50;
    }

Exercice 3 - Vignettes

  2. Faites en sorte que toutes les images à l'intérieur des boites (pas les autres) aient une largeur (width) de 180 pixels.
    div img {
     width: 180px;
    }
  3. Au lieu d'afficher les images verticalement, on souhaite les afficher horizontalement. Utilisez pour cela l'attribut float.
    div img {
     width: 180px;
     float: left;
    }
  4. Que se passe-t-il si l'on fait varier la taille de la fenêtre du navigateur horizontalement ?
    Lorsque nous faisons varier la taille du navigateur horizontalement les images s'alignent afin d'occuper au mieux l'espace disponible, quitte à passer à une nouvelle ligne.
  5. Vérifiez que vous pouvez aussi mettre toutes les images à droite en changeant juste un mot dans la partie CSS du document. Puis remettez les images alignées à gauche.
    div img {
     width: 180px;
     float: right;
    }
  6. Faites en sorte que les titres (animal1, animal2, etc.) soient affichés en blanc sur fond noir et soient centrés horizontalement par rapport à la boite qui les contient. Jouez de plus avec la propriété padding pour avoir un espacement convenable.
    div div {
     padding-top: 5px;
     padding-bottom: 5px;
     background-color: black;
     color: white;
     text-align: center;
    }
  7. Espacez les unes des autres les boites contenant les images avec la propriété margin.
    div {
     ...
     margin: 1em;
    }
    div div {
     ...
     margin: 0;
    }
  8. Faites en sorte que les images à l'intérieur des boites aient une bordure d'un pixel de largeur de couleur #aaaaaa.
    div img {
     width: 180px;
     border: solid #aaaaaa;
     border-width: 1px;
    }
  9. Changez l'opacité (propriété opacity) lorsque l'utilisateur passe sa souris au-dessus d'une image dans une boite.
    div img:hover {
     opacity: 0.5;
    }
  10. Faites en sorte que les titres soient accolés par le bas à l'image et mordent légèrement sur l'image (de 5 pixels par exemple).
    div {
     ...
     position: relative;
    }

    div div {
     ...
     top: -5px;
    }

Exercice 4 - Barre de menu

  1. Faites en sorte que la liste ait une position fixe, en haut à gauche de la partie affichage du navigateur.
    ul {
     position: fixed;
     top: 0;
    }
  2. Retirez le style par défaut des items de liste (avec list-style-type) pour qu'il n'y ait pas de style par défaut.
    li {
     list-style-type: none;
    }
  3. Affichez les items en gras et avec une police Arial (ou sinon sans-serif).
    li {
     ...
     font-weight: bold;
     font-family: Arial, Helvetica, sans-serif;
    }
  4. Faites en sorte que les items soient affichés horizontalement collés à gauche et non plus verticalement.
    li {
     ...
     float: left;
    }
  5. Faites en sorte que les items aient une largeur de 100 pixels et ajoutez un espacement d'une dizaine de pixel entre les items.
    li {
     ...
     width: 100px;
     margin: 10px;
    }
  6. Changez le fond du menu pour qu'il soit de couleur jaune (yellow), en chantant yellow submarine, et changez la couleur des items pour qu'ils soient gris foncés.
    ul {
     ...
     background-color: yellow;
    }

    li {
     ...
     color: darkgray
    }
  7. Fixez la hauteur (height) du menu à 50 pixels puis faites en sorte que le menu occupe toute la largeur (width).
    ul {
     ...
     height: 50px;
     width: 100%;
    }
  8. Faites en sorte que l'item 'About' soit affiché à droite de la barre de menu.
    li:last-child {
     float: right;
    }
  9. Affichez une ligne de 2 pixels en bas du menu.
    ul {
     border-bottom: solid;
     border-width: 2px;
    }

Exercice 5 - Faire un sous-menu (Optionnel)

  1. Positionnez le sous-menu juste en dessous de l'item web (avec relative/absolute).
    ul ul {
     position: relative;
     top: 25px;
    }
  2. Utilisez la propriété display pour ne pas afficher le sous-menu par défaut.
    ul ul {
     ...
     display:none;
    }
  3. Faites en sorte d'afficher le sous-menu si l'on passe la souris sur l'item 'Web'.
    ul li:hover ul {
     display: block;
    }
  4. N'oubliez pas de jouer avec la propriété z-index pour mettre le sous menu au-dessus du reste de la page.
    ul ul {
     ...
     z-index: 9999;
    }
