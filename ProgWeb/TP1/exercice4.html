Exercice 4 - Service HTTP

  1. Utilisez nc pour vous connecter au serveur Web de la machine monge.univ-mlv.fr en utilisant le port TCP correspondant au protocole HTTP puis tapez la requête suivante :
    GET /ens/ HTTP/0.9
    Header de la réponse:
    HTTP/1.1 200 OK
    Date: Thu, 28 Nov 2019 09:08:15 GMT
    Server: Apache
    Content-Length: 2957
    Connection: close
    Content-Type: text/html; charset=UTF-8
  2. A quoi correspondent les 3 valeurs sur la première ligne de l'entête ?
    La première ligne signifie que la requête s’est bien déroulée, on y voit le protocole, sa version et le code de retour de la requête.
    La deuxième ligne est la date à laquelle la requête à été lancée.
    La troisième est le nom du serveur web utilisé.
  3. Quel est le nom du serveur Web ?
    Le nom du serveur web est Apache, le plus utilisé.
  4. A quoi correspond la valeur de Content-Type ?
    La valeur du content-type correspond au type de fichier auquel on accède ainsi que l’encodage des caractères sur le dit fichier.
  5. A quoi correspond la valeur de Content-Length ?
    La valeur du content-length correspond à la taille du fichier auquel on accède.
  6. Faites la même analyse mais avec la requête:
    GET /ens/toto.txt HTTP/0.9
    Réponse:
    HTTP/1.1 404 Not Found
    Date: Thu, 28 Nov 2019 10:09:16 GMT
    Server: Apache
    Connection: close
    Content-Type: text/html
    La page est inexistante
  7. Enfin, faites la même analyse avec la requête:
    GET /index.html HTTP/0.9
    Réponse:
    HTTP/1.1 302 Found
    Date: Thu, 28 Nov 2019 10:10:12 GMT
    Server: Apache
    Location: http://igm.u-pem.fr/index.html
    Content-Length: 279
    Connection: close
    Content-Type: text/html; charset=iso-8859-1
    La page a été déplacée
