Exercice 3 - Perroquet, quelle heure est-il ?

  1. On cherche à savoir à quoi correspond le fichier /etc/services.
    Le fichier /etc/services contient un libellé des services internet avec les ports et protocoles utilisés.
  2. Rappelez ce que fait la commande grep. Utilisez la commande grep pour déterminer le numéro du port TCP correspondant au service echo.
    La commande grep permet de rechecher dans un fichier une suite de caractères passée en argument
    La commande grep echo /etc/services renvoie le port 7 pour le protocole TCP de la commande echo.
    saravanane@PC:~$ grep echo /etc/services
    echo 7/tcp
    echo 7/udp
  3. Utilisez la commande nc pour tester le service echo sur la machine gaspard.univ-mlv.fr.
    saravanane@PC:~$ nc gaspard.univ-mlv.fr 7
    Perroquet
    Perroquet
  4. Faites la même manipulation avec le service daytime toujours sur la machine gaspard.univ-mlv.fr. Comment interpréter la réponse de la machine ?
    Le résultat correspond à l’heure de la machine à laquelle on accéder au moment de la requête
    saravanane@PC:~$ grep daytime /etc/services
    daytime 13/tcp
    saravanane@PC:~$ nc gaspard.univ-mlv.fr 13
    27 NOV 2019 17:12:35 CET
  5. Enfin, quel est le port TCP correspondant au service HTTP ?
    Le port TCP correspondant au service HTTP est le 80.
    saravanane@PC:~$ grep HTTP /etc/services
    http 80/tcp www # WorldWideWeb HTTP
